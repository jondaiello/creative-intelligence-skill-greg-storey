# A GPT Skill "Creative Intelligence" by Greg Storey
A starting point for creative thinking skills to be used with popular GPTs based on Greg Storey's book "Creative Intelligence"

## Instructions
1. Ensure you have a [paid subscription to Claude](https://claude.com/pricing)
2. Download the [.skill file](creative-intelligence.skill)
3. Install the .skill file [in your account](https://claude.ai/settings/capabilities). You'll find it under your Account Settings > Capabilities, scroll to the "Skills" section.
4. Use it in your next Claude conversation

Book available at: https://brilliantcrank.com/creative-intelligence/
